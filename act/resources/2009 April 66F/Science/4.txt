<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">﻿</div>

<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">The octane number of a fuel is a measure of how smoothly the fuel burns in a gasoline/engine. Lower octane fuels knock (explode) when burned, which lowers fuel efficiency and can cause engine damage. Heptane knocks considerably when burned and is given an octane number of 0. Isooctane knocks very little and is given, an octane number of 100.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Different proportions of heptane and isooctane were mixed to obtain mixtures with octane numbers between 0 and 100 (see Table 1).</div>
