
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">G. soja (a wild soybean) produces y-tocopherol (a type of vitamin E). It then converts some y-tocopherol into α-tocopherol (another type of vitamin E). In G. soja, the enzyme TMT catalyzes this reaction:</div>

<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;"><img src='file:\$$$\resources\2008 April 65D\imgs\158.png' width = '319'height = '36'/></div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">A. thaliana (a mustard plant), produces y-tocopherol, but lacks TMT, so it produces only a small amount of α-tocopherol.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Because α-tocopherol is more effective at reducing cellular damage than is y-tocopherol, a scientist tried to transfer G. soja's TMT gene into A. thaliana.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Experiment 1</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Four versions of G. soja's TMT gene (TMT1-TMT4) were cloned. Each gene was incorporated into a vector (a biological structure that carries a gene and transfers it into the cells of an organism).</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Six genetically identical lines of A. thaliana were developed (L1-L6). As shown in Table 1, L1-L4 were each exposed to a vector carrying 1 of the 4 cloned genes; L5 was exposed only to the vector; and L6 was left untreated.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Next, 10 plants from each line were grown. Table 1 gives the tocopherol concentration and the percent (%) by mass of the 2 types of tocopherol in the plants.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Experiment 2</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Four genetically different strains of A. thaliana were grown (S1-S4). S1-S4 were each exposed to a vector carrying TMT1.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Next, 10 plants from each strain were grown. Table 2 shows the tocopherol concentration and the percent by mass of the 2 types of tocopherol in the plants.</div>
