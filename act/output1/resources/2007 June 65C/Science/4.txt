
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Amylase is an enzyme that speeds up the breakdown of starch into smaller carbohydrates (such as maltose).</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">A starch solution, an amylase solution, and several buffer solutions (solutions that maintain a constant pH) were prepared. Two experiments were done using these solutions and the following tests:</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">• Iodine3 drops of a brown <img src='file:\$$$\resources\2007 June 65C\imgs\226.png' width = '41'height = '38'/> solution is added to a sample solution. If starch is present in the sample solution, the solution will turn from brown to black.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">• Benedict's test: 1 mL of a blue solution containing <img src='file:\$$$\resources\2007 June 65C\imgs\227.png' width = '75'height = '38'/> is added to a sample solution. The solution is then heated in a boiling water bath for 3 min. If maltose is present in the sample solution, the solution will turn from blue to orange.</div>

<br>
<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Experiment 1</div>

<div class = "para" style = "font-size: 16;font-family:Microsoft YAHEI;">Each of 7 test tubes was filled with 5 mL of pH 7.0 buffer solution. Different amounts of starch solution and/or amylase solution were added to each tube. The solution in each tube was then diluted to 10 mL with the buffer solution (so the final pH was 7.0 in each tube). Each tube was then incubated at a constant temperature for 10 min. The iodine and Benedict's tests were then each done with 2 mL samples drawn from each tube (see Table l).</div>
